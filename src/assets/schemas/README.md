These schemas are trimmed, self-contained snapshots of the SchemaStore `package.json` schema (`https://www.schemastore.org/package.json`) and its cataloged Composer schema (`https://getcomposer.org/schema.json`). They retain the documented top-level properties, types, enums, descriptions, and common nested shapes used for completion and validation. Remote tool-configuration references and deep repository/package-manager definition chains were replaced with permissive inline object shapes so Monaco never needs a schema request service.
